There are these types of files: .html .js .css .php and .json

.html: a languege defines the framework for client side, can contain js php and css;

.css: a language defines the style of client side

.js: language that define all the functions, such as button, ajax; 

Jquery is an external js library; kind like opencv to python;
 ajax is js library build into browser, also included in jquery; 
 jquery and ajax works together to send and read data to and from server;

Bootstrap is a combination of external css and js library for client side;

.php: the server side laguage; 
      it process all server side comunication and process;

XML: it technically a type of file, but it is generated by server components.
